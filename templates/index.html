Catálogo PDF

Generador de Catálogos PDF

Crea catálogos profesionales desde una carga manual o importando tus productos desde Excel.

Organiza tus productos, agrega imágenes y genera un PDF listo para compartir con tus clientes.

¿Cómo funciona la aplicación?

1. Elegí el método de carga

La aplicación permite trabajar de dos maneras:

  • Carga Manual: Agregás productos uno por uno.
  • Carga por Excel: Importás muchos productos automáticamente desde un archivo Excel.

2. Configurá las columnas del Excel

Si utilizás Excel, podrás indicar qué significa cada columna de tu archivo.

Ejemplo:

  • Producto → Nombre
  • Costo → Precio
  • Detalle → Descripción

También podrás crear campos personalizados como:

  • Marca
  • Color
  • Talle
  • Stock

3. Administrá tus productos

Desde la lista de productos podrás:

  • Agregar productos
  • Editar información
  • Eliminar productos
  • Agregar imágenes
  • Agregar nuevos campos

4. Validación antes del PDF

Antes de generar el catálogo, la aplicación verificará qué productos no tienen imagen.

Podrás:

  • Editar el producto
  • Agregar una imagen
  • Eliminar productos incompletos

5. Generá tu catálogo PDF

Finalmente podrás generar un catálogo PDF profesional con todos tus productos.

Ideal para:

  • Ventas
  • Mayoristas
  • Redes sociales
  • WhatsApp
  • Presentaciones comerciales
